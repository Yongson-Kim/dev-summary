# Django 개발환경 설정

## 가상환경 생성

```bash
$ mkdir django_project_name
$ cd django_project_name
django_project_name $ python3 -m venv venv_name
django_project_name $ source ./venv_name/bin/activate
(venv_name) django_project_name $
```

## Django 설치

**가상환경**에서 실행합니다.

```
(venv_name) django_project_name $ 
```
